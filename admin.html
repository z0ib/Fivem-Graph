Connexion GitHub

Entre ton token GitHub et le nom de ton repo. Ces infos sont sauvegardées uniquement dans ton navigateur (localStorage), elles ne partent nulle part.

Publier le site

Génère ton index.html et pousse-le directement sur GitHub. GitHub Pages redéploie automatiquement en ~2 minutes.

Comment créer ton token GitHub

1. Va sur github.com/settings/tokens

2. Clique Generate new token → classic

3. Donne-lui un nom (ex: admin-packs)

4. Coche uniquement repo (toute la section)

5. Clique Generate token et copie-le ici

⚠️ Le token ne s'affiche qu'une seule fois sur GitHub — copie-le tout de suite !
